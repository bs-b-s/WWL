## REST스러운 것
잘 알려져 있지만 뭔가 미묘하게 잘 모르는, 팀 프로젝트를 하는 상황이고 코드 리뷰 등을 할 때 
> "그 부분은 뭔가 REST 스럽지 않은데요?"

라고 말하면 이런게 **"REST API다"** 라고 반박은 해주고 싶은데 본인도 잘 모르는 경우가 많다.

이렇게 자신이 매일 개발하는 API의 정확한 정의와 사용법을 알아둘 필요가 있다..
